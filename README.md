# den

A (S)SSG written in Go

## Usage

    den add post_src
    den update [id, post_src]
    den rm id

## Dependencies

 * [go-sqlite3](https://github.com/mattn/go-sqlite3)
 * [goldmark](https://github.com/yuin/goldmark)
